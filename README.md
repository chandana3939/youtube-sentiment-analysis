 ## YouTube Comment Sentiment Analysis

This project analyzes the sentiment of YouTube comments using Python and the YouTube Data API.

# Technologies Used
- Python
- YouTube Data API
- TextBlob
- Pandas
- Matplotlib

# Features
- Extracts comments from a YouTube video
- Performs sentiment analysis (Positive, Negative, Neutral)
- Saves results to CSV file
- Visualizes sentiment distribution

# Files
- youtube_sentiment.py → Python script for analysis
- youtube_comments_sentiment.csv → Output dataset

# Output:
The program classifies comments and generates a sentiment dataset for further analysis.

Note: The YouTube API key is not included in this repository for security reasons. Please generate your own API key from the Google Cloud Console and replace YOUR_API_KEY in the code before running the project.
